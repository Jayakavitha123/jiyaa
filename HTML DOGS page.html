HTML BASICS

Tags

The basic structure of an HTML document includes tags, which surround content and apply meaning to it.

Closing tags

Not all tags have closing tags like this some tags,which do not wrap around content will close themselves.

The body and html tags put close to their respective elements.

Attributes

Tags can also have attributes, which are extra bits of information.

Attributes appear inside the opening tag and their values sit inside quotation marks.

They look something like <tag attribute="value">Margarine</tag>. We will come across tags with attributes later.

Elements

Tags tend not to do much more than mark the beginning and end of an element.

Elements are the bits that make up web pages.

Page titles

All HTML pages should have a page title.

Title should inside the head element of opening and closing tags.

The body element contains information about the page.

Paragraphs

The ptag is used for paragraphs.

If we want to appear on different lines,we have to use p tag.

Emphasis

When paragraphs using emand strong,browsers will display em in italics and strong in bold.

Line breaks

Line breaks is also used to separate lines.

Lists

There are three types of lists;

  • unordered lists
  • ordered lists
  • definition lists

The ul and ol lists work in the same way, non-sequential lists usually proceded by bullets.The sequential lists are normally represented by numbers.

The li tag is used to define each list item.

Links

An Anchor(a) is used to define a link.
The destination link is defined in href attribute of the tag.
